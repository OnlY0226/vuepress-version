                                  # 基础属性管理

                                  由于 文档类型 的 基础属性 较多，部分 文档类型 的 基础属性 不宜显示，所以需配置显示的 基础属性，以便管理文档 基础属性。

                                  • 点击 新建按钮 新建需要设置的文档类型
                                  • 点击列表设置文档类型
                                  • 点击显示模块的 新建按钮 添加需要显示的 基础属性（1）。
                                  • 点击显示模块的 显示列 的 开关 控制 基础属性 显隐（2）。
                                  • 点击显示模块的 操作列 的 删除 删除显示的 基础属性（3）。
                                  • 点击显示模块列表编辑 基础属性。
                                  • 点击相关文档模块的 新建按钮 添加需要相关联的 文档类型（4）。
                                  • 点击相关文档模块的 操作列 的 删除 删除相关联的 文档类型（5）。
