                                  # 工作流程管理

                                  为便于管理，可在此管理工作流程。 对工作流程进行重分配与删除。

                                  • 点击操作列的 删除按钮 删除工作流。（1）
                                  • 点击操作列的 指派按钮 重新指派工作流代理人。（2）
                                  • 点击顶栏 删除按钮 批量删除工作流，需选择工作流。（3）
